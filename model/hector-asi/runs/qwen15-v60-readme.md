# Qwen15 V60 KTO

Este experimento prueba KTO prospectivo con referencia V41 congelada. No sustituye entrenamiento por reglas ni prompts. Los pesos se generan fuera de Git mediante dos réplicas CPU deterministas y solo podrán promoverse con mejora conductual estricta, seguridad crítica perfecta, límites de longitud/latencia y archivos byte-idénticos.
