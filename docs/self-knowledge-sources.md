# Fuentes del núcleo de autoconocimiento

Verificado el 21 de julio de 2026.

El manifiesto `worker/intelligence/product-knowledge.ts` separa hechos estables, estado operativo y datos temporales. Los datos temporales no deben tratarse como verdad permanente: Héctor OS debe volver a consultar fuentes oficiales cuando una respuesta dependa de disponibilidad, precios, límites, modelos o funciones actuales.

Fuentes oficiales de referencia:

- OpenAI Help Center — ChatGPT Work and Codex: https://help.openai.com/en/articles/20001275
- OpenAI Help Center — Tasks in ChatGPT: https://help.openai.com/en/articles/10291617-tasks-inchatgpt
- OpenAI — Introducing the Codex app: https://openai.com/index/introducing-the-codex-app/
- OpenAI — GPT-5.6: https://openai.com/index/gpt-5-6/

Regla de evidencia: el nombre solicitado por el usuario o escrito en un prompt no demuestra el modelo usado. Para afirmar el modelo efectivo se requiere el identificador devuelto por el proveedor o telemetría/configuración comprobable de la ejecución.
