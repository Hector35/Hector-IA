# Free-first integration checklist

- [x] Registro de recursos gratuitos y licencias.
- [x] Gobernador de llamadas compartido con el runtime.
- [x] Una llamada avanzada por defecto incluso con razonamiento alto.
- [x] Múltiples inferencias únicamente por petición explícita.
- [x] Caché exacta por usuario, verificada y con expiración.
- [x] Recuperación semántica existente limitada a evidencia verificada.
- [x] Compactación y deduplicación de contexto.
- [x] Métricas persistentes de llamadas avanzadas, gratuitas, tokens y costo evitado.
- [x] Bloqueo de alto riesgo sin autorización.
- [x] Benchmark reproducible versionado.
- [ ] Repository PR Checks aprobados.
- [ ] Production Audit aprobado.
- [ ] Migraciones D1 comprobadas por CI.
- [ ] PR fusionado y `main` verificado.
